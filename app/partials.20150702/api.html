RESTful API

HipSci has a RESTful API which provides programmatic access to the project's data.

The RESTful API is built on Elasticsearch, and uses many of the options described in the Elasticsearch documentation.

Endpoints

Information about a cell line: http://www.hipsci.org/lines/api/cellLine/:name

  • E.g. GET http://www.hipsci.org/lines/api/cellLine/HPSI0813i-fpdj_3
  • Returns information in json format
  • The cellLine data structure is defined by this mapping. See this guide for information on Elasticsearch mapping definitions

Information about a donor: http://www.hipsci.org/lines/api/donor/:name

  • E.g. GET http://www.hipsci.org/lines/api/donor/HPSI-ffdj
  • Returns information in json format
  • The donor data structure is defined by this mapping. See this guide for information on Elasticsearch mapping definitions

Search for cell lines: http://www.hipsci.org/lines/api/cellLine/_search

  • This endpoint supports any search option described in the Elasticsearch documentation on Full Body Search
  • Returns information in json format
  • E.g. POST http://www.hipsci.org/lines/api/cellLine/_search '{"query": {"match": {"sourceMaterial.cellType": "Fibroblast"}}}'

Search for donors: http://www.hipsci.org/lines/api/donor/_search

  • This endpoint supports any search option described in the Elasticsearch documentation on Full Body Search
  • Returns information in json format
  • E.g. POST http://www.hipsci.org/lines/api/donor/_search '{"query": {"match": {"age": "20-24"}}}'
